HTML Formatting Tags

  • Bold text - Use <b> or <strong> tags
  • Italic text - Use <i> or <em> tags
  • Underline text - Use <u> tag
  • Small text - Use <small> tag
  • Large text - Use <font size> attribute

HTML Structure Elements

  • Headings - <h1> to <h6>
  • Paragraphs - <p>
  • Line breaks - <br>
  • Horizontal rules - <hr>
